## Sobre mí

Soy una profesional con amplia experiencia en gestión de servicios, operaciones, atención al cliente y recursos humanos, actualmente enfocada en el desarrollo web, la programación orientada a objetos y las bases de datos relacionales.

Mi trayectoria me ha permitido desarrollar una visión práctica de negocio, coordinación de equipos, análisis de procesos y resolución de incidencias. Ahora aplico esa experiencia al desarrollo de soluciones tecnológicas útiles, organizadas y orientadas a mejorar la eficiencia y la experiencia de usuario.

- Actualmente refuerzo mis conocimientos en desarrollo web y bases de datos.
- Me interesan los proyectos con HTML, CSS, JavaScript, PHP, Python y SQL.
